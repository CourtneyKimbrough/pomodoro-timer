# Pomodoro Timer
## This is a console pomodoro timer.
[![Java](https://img.shields.io/badge/Java-%23ED8B00.svg?logo=openjdk&logoColor=white)](#)

## How it works 
The pomodoro timer follow this pattern:
 - 25 minutes of work
 - 5 minute break
 - 25 minutes of work
 - 5 minute break
 - 25 minutes of work
 - 5 minute break
 - 25 minutes of work
 - 20 minute break
